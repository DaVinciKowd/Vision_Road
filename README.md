# Software

The software developed for this study is titled **Vision Road**.  


## Project Description

It serves as an interface for recording and monitoring road hazards, integrating real-time hazard detection based on the proposed system.
It aims to focus on road surface hazards on the Southern Tagalog Arterial Road (STAR) Tollway
